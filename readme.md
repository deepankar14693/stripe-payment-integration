**stripe-payment-integration using nodejs express and stripe**

*A simple demo project for setting up stripe payments with nodejs express and stripe(includes only charging)*


**TECHNOLOGIES**

  **back-end**
  
  \* nodejs
  \* express
  \* stripe
  
  **front-end**
  
  \* reactjs
  \* StripeCheckout component(npm)
  \* axios
